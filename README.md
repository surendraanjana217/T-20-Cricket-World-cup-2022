# T20 World Cup Data Analytics Project

## Overview
This project analyzes T20 World Cup cricket data using Python, Pandas, NumPy, and Power BI.

## Tools Used
- Python
- Pandas
- NumPy
- Power BI

## Key Analysis
- Top Run Scorers
- Top Wicket Takers
- Strike Rate Analysis
- Batting Performance
- Bowling Performance
- Interactive Dashboard

## Outcome
The project transforms raw cricket data into meaningful insights through data cleaning, analysis, and visualization.
